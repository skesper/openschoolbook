\chapter{Funktion}

Der Begriff einer Abbildung ist uns schon Ã¶fter untergekommen. Eine Funktion ist eine Abbildung, die zwei Mengen (hier $X$ und $Y$) in Beziehung setzt:

\[ f: X \longrightarrow Y \]
$f$ ordnet jedem Element $x\in X$ ein Element $y\in Y$ zu, indem
\[f(x) =y\]
gilt. Die Umkehrung gibt es allgemein nicht, aber wenn sie existiert, so wird sie als $f^{-1}$ bezeichnet und es gilt:
\[ f^{-1}(y) =x\]
